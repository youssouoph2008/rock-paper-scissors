# ✊✋✌️ Rock, Paper, Scissors

A simple web application of the classic Rock, Paper, Scissors game, built with HTML and JavaScript. Play against the computer and test your luck 🤖

# 🚀 Demo

You can play the game by simply opening the index.html file in your browser.

# Technologies

HTML 5
JavaScript

# 📌 Features
Simple and user-friendly interface
Play against the computer
Instant results
Game logic implemented in JavaScript
Great for beginners in web development
🛠️ Technologies Used
HTML5
JavaScript (Vanilla JS)

📁 rock-paper-scissors

▶️ How to Use

Clone this repository or download the files:

git clone `https://github.com/youssouph2008/rock-paper-scissors.git`
Open the index.html file in your browser.
Choose one option:
✊ Rock
✋ Paper
✌️ Scissors
See if you can beat the computer!
🧠 Game Logic
Rock beats Scissors
Scissors beat Paper
Paper beats Rock
If both choices are the same → It's a draw

The computer makes a random choice using JavaScript.
